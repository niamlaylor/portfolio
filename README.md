## Getting Started

First, run the development server:

```bash
npm run dev
# you can also use 
# yarn or pnpm
```

Then install all dependencies:

```bash
npm install
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the local version of the project.

## Notable Dependencies
- font awesome
- autoprefixer
- axios
- eslint
- next
- react
- tailwind
- xml2js
